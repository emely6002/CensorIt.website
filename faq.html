FAQ

Short answers to common questions.

Who is CensorIt for?

Content creators who plan to go live in outdoor areas and need to censor sensitive material or individuals faces.

How long does installation usually take?

Installation is quick and easy. From downloading the app to streaming with it, it takes around 5 minutes.

Can CensorIt run on-premises?

Many teams start with a local or private-network deployment for policy testing. Your deployment options depend on the edition you choose—check the install guide for the path that matches your stack.

What do I need before installing?

The repository (or release package), a supported runtime if your edition requires one, and a clear owner for policy configuration and access control.

Is this website open source?

This site is a simple static template you can copy and adapt. See How to install for running it locally.
