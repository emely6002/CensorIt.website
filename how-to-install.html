How to Install

The CensorIt marketing site is a static bundle (HTML, CSS, and a small script). There is no package manager required. Pick the path that matches how you work.

  1. Get the files

    Clone the repository or download the folder that contains index.html and the styles.css / script.js assets. 

    git clone <your-repo-url>
cd business-website
  2. Open the site

    Double-click index.html to open it in your default browser, or drag the file into an open browser window.

  3. Optional: local server

    For routing and caching behavior closer to production, serve the folder with any static file server. For example, with Python: 

    python -m http.server 8080

    Then visit http://localhost:8080 in your browser.
