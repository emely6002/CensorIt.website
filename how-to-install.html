How to Install

The following is the list of steps in order to download the plug-in and launch it in OBS.

  1. Download Files (Windows)

    Go to the following link. https://github.com/FelipeReyesSag/obs-backgroundremoval.git

  2. Unzip the Files

    On Windows -> Right click File Explorer, go to Downloads, and right-click the Zip File.

  3. Drag Files to OBS

    On Windows -> Find the .dll file and go to the OBS file directory. Go into the OBG plug-in folder and paste.
