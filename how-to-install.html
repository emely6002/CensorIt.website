How to Install

The CensorIt marketing site is a static bundle (HTML, CSS, and a small script). There is no package manager required. Pick the path that matches how you work.

  1. Download Files (Windows)

    Click the following link. 

    git clone <your-repo-url>
cd business-website
  2. Unzip the Files

    On Windows -> Right click File Explorer, go to Downloads, and right-click the Zip File.

  3. Drag Files to OBS

    On Windows -> Find the .dll file and go to the OBS file directory. Go into the OBG plug-in folder and paste. 

    python -m http.server 8080
